# 游戏特性设计文档 (Game Features Design)

本文档描述了游戏的核心玩法特性和辅助功能。

## 1. 网格地图与移动 (Grid Map & Movement)

- **核心界面**: 游戏的主战斗区域是一个 4x4 的网格。
- **内容填充**: 每次“搜寻”后，网格上的15个非玩家地块会根据以下概率独立填充：
  - **怪物 (Monster)**: 70%
  - **空格 (Empty)**: 15%
  - **宝箱 (Chest)**: 5%
  - **体力药水 (Item)**: 5%
  - **怪物镇守的宝藏 (Guarded Treasure)**: 5% (每张图最多一个)
- **状态持久化**: 地图状态（包括所有敌人、宝箱和物品的位置）会随着游戏进度自动保存。刷新页面**不会**重置地图，此举旨在防止玩家利用刷新来规避“敌人主动进攻系统”的惩罚。
- **移动与互动规则 (`isPathPossible`)**:
    - **路径判定**: 玩家的移动基于广度优先搜索算法（BFS）。
    - **可通行地块**: `empty`, `player`, `item` (体力药水), 和 `guarded_treasure` (怪物镇守的宝藏) 类型的地块被视为可通行。`monster` 和 `chest` 是障碍物。
    - **互动**: 玩家可以与**紧邻**（上下左右）其可通行范围的任意地块进行互动。这意味着，只要玩家能走到某个怪物或宝箱的旁边，就可以对其进行操作。

## 2. 楼层系统 (Floor System)

楼层系统是游戏后期内容的核心，为玩家提供了不断挑战自我、追求更高品质战利品的目标。

### 2.1 楼层控制

在战斗区域下方，有一个专用的楼层控制面板，包含以下功能：
- **`当前楼层`**: 显示玩家目前所在的楼层。
- **`上一层`**: 点击后返回上一层，并刷新地图。在第1层时此按钮会被禁用。
- **`下一层 / 挑战更高层`**:
  - 如果当前楼层**低于**玩家已达到的最高楼层，此按钮为“下一层”，点击后会直接前进一层并刷新地图。
  - 如果当前楼层**等于**玩家已达到的最高楼层，此按钮变为“挑战更高层”，点击后将开启楼层晋升挑战。
- **`快速前往`**: 点击后弹出一个窗口，允许玩家输入一个已解锁的楼层号，直接跳转到该楼层。

### 2.2 楼层效果

当前所在的楼层会极大地影响怪物和装备的生成概率，核心规则是**“楼层越高，下限越高”**。
- **装备品质影响**: 每提升一层，“普通”品质装备的基础出现概率降低1%。
- **怪物境界影响**: 每提升一层，最低级境界“下级佣兵”的出现概率降低1%。
- **概率再分配**: 减少的低级概率会被系统自动、按比例地分配给更高等级的品质和境界，使得在高楼层中获得高级战利品和遭遇强大敌人变得更加普遍。

### 2.3 楼层挑战

- **目的**: 解锁并前往一个全新的、更高的楼层。
- **挑战内容**: 系统会生成一个特殊的挑战地图，该地图上仅有**一个“远古”阶级**的怪物。该怪物的境界被锁定为**玩家当前楼层所能生成的最低境界**。
  - **示例**: 玩家在第80层，挑战第81层。由于在80层时，“下级佣兵”的出现概率已降为0，新的最低境界变为“中级佣兵”。因此，挑战内容就是击败一个“远古级”的“中级佣兵”。
- **胜利条件**: 玩家只需**击败地图上的那个怪物**即可完成挑战。
- **奖励**: 挑战成功后，玩家的“最高楼层”记录会+1，并自动前进到这个新的楼层。

## 3. 自动化系统 (Automation System)

玩家可以通过点击战斗区域右上角的“自动设置”按钮来配置自动化功能。该面板包含以下独立开关和一个总开关。

- **自动化循环 (Automation Cycle)**:
    - 当任何自动功能开启时，系统会以固定的0.5秒为间隔进入一个循环。在一个循环（或“一帧”）内，系统会：
    1.  扫描整个游戏状态，找出**所有**符合当前自动化设置的可执行操作。
    2.  将这些操作（如清缴多个0伤怪、拾取多个宝箱、淬炼一次装备）作为一个批次，**在一次更新中全部执行**。
    3.  所有战斗的奖励（修为、战利品）和拾取的物品会先被收集，然后统一进行结算。这意味着，如果一次自动循环击败了3个提供经验的怪物，玩家的经验会一次性增加，并可能触发连续升级。
    4.  **视觉反馈**: 自动化批处理完成后，所有产生的反馈信息（如经验、战利品、体力恢复）会像手动操作一样，在对应的地块上以**序列化**的方式依次弹出，确保了视觉体验的一致性。
    - 这个批处理机制确保了自动化的效率和结果的准确性，避免了逐个处理可能导致的计算错误。

- **`自动清0伤怪 (Auto Fight)`**:
    - **触发条件**: 在自动化循环中，系统会找出**所有**满足以下条件的怪物并自动攻击：
        1. 战斗模拟结果为玩家胜利 (`combatOutcome.playerWon`)。
        2. 战斗后的体力变化大于或等于0 (`combatOutcome.healthChange >= 0`)。
        3. 该怪物地块对于玩家是路径可达的 (`isPathPossible`)。

- **`自动拾取 (Auto Loot)`**:
    - **触发条件**: 在自动化循环中，系统会自动打开**所有**路径可达的 `chest` (宝箱) 和拾取**所有**路径可达的 `item` (体力药水)。
    - **特殊规则**: 对于“怪物镇守的宝藏”，只有当其**被解锁后**（即地图上所有怪物和宝箱被清除），“自动拾取”才会尝试获取它。

- **`自动装备 (Auto Equip)`**:
    - **适用范围**: 此逻辑同时应用于“一键换装”按钮和自动化战利品处理流程。
    - **优先级规则**: 比较装备时，严格遵循以下顺序：
        1.  **高境界**: 装备的境界阶层 (`佣兵`, `职业者`, `英雄`等) 更高者胜出。
        2.  **神性词条**: 在同一境界阶层下，拥有`神性`词条的装备优先于没有的。如果两件装备都拥有`神性`，则该词条数值更高者胜出。
        3.  **高品质**: 在前序条件都相同的情况下，装备品质 (`传说`, `史诗`等) 更高者胜出。
        4.  **高战力**: 在前序条件均相同的情况下，战力更高者胜出。
    - **虚空装备特殊规则**: 当获得的新装备和对应槽位的已装备物品**均为虚空品质**时，系统不会执行常规的替换逻辑，而是会触发上文《装备系统》中描述的**“吞噬”进化机制**。

- **`自动炼化 (Auto Refine)`**:
    - **触发时机**: 当通过`自动拾取`或`战斗掉落`获得新装备时，此功能会与`自动装备`协同工作。
    - **协同逻辑**:
        1.  **装备槽为空**:
            - `自动装备`开启 -> **装备**新物品。
            - `自动装备`关闭 -> 新物品进入背包。
        2.  **装备槽有物品**:
            - **新物品是“升级”**:
                - `自动装备`开启 -> **装备**新物品。被替换下的旧物品，如果`自动炼化`开启则**炼化**，否则进入背包。
                - `自动装备`关闭 -> 新物品进入背包（防止错过升级）。
            - **新物品非“升级”**:
                - 如果`自动炼化`开启，且该新物品**不是虚空品质**，则**炼化**该新物品。
                - 如果该新物品是**虚空品质**，或者`自动炼化`关闭，则新物品进入背包。虚空装备永远不会被自动炼化。

- **`自动淬炼 (Auto Temper)`**:
    - **触发条件**: 在自动化循环中，如果玩家拥有淬炼石 (`temperingStones > 0`)，系统会执行**一次**淬炼操作。
    - **目标选择逻辑**:
        1.  **扫描范围**: 系统会扫描**已装备**和**背包中**的所有物品。
        2.  **优先级排序**: 在所有可淬炼的装备中，按照 `自动装备` 的优先级（境界 > 品质 > 战力）进行排序。
        3.  **选择目标**: 自动选择排序后最顶尖的一件装备。
        4.  **选择词条**: 在选定的装备上，自动选择**完美度最低**的词条进行淬炼。
        5.  **循环**: 每次获得新的淬炼石，自动化循环都会在下一帧尝试执行此逻辑。
    - **无确认**: 淬炼过程静默进行，无任何确认提示。淬炼结果（包括具体的属性提升数值）会直接显示在游戏日志中。

- **`一键开启/关闭`**: 在“自动设置”面板的底部，提供一个总开关，可以方便地同时启用或禁用上述所有五项自动化功能。

## 4. 挑战系统 (Challenge System)

### 4.1 境界挑战

- **目的**: 解锁在“搜寻限制”中搜寻特定高阶敌人的能力。
- **流程 (`handleInitiateChallenge`)**:
    1.  玩家在“搜寻设置”面板中，点击一个未解锁的怪物阶级。
    2.  游戏进入“挑战模式”，并生成一个布满该指定阶级和玩家当前境界的特殊地图。
    3.  玩家只需 **击败地图上的任意一个怪物** 即可完成挑战。
    4.  挑战成功后，该怪物阶级被永久解锁，并退出挑战模式。

### 4.2 放弃挑战

- **功能**: 在任何挑战（境界挑战或楼层挑战）期间，地图上方的“搜寻”按钮会变为红色的“放弃挑战”按钮。
- **效果**: 点击后，玩家会退出当前的挑战模式，地图会立即刷新为常规的探索地图。
- **风险**: 如果玩家的“敌人主动进攻系统”已激活，放弃挑战后**依然会像正常搜寻一样受到一次攻击**，需要谨慎使用。

## 5. 信息记录 (Top Stats Panel)

- **功能**: 一个侧边栏，用于记录玩家在整个游戏过程中的各项“之最”数据。
- **记录内容** (每个列表最多记录10项):
    - **战力最高装备**
    - **品质最高装备** (按品质等级+虚空重数排序，同级按战力排序)
    - **遭遇过的最强敌人** (按战力排序)
    - **体力恢复最多敌人** (按单次战斗胜利后恢复的体力值排序)

## 6. 炼化 (Refining)

- **功能**: 玩家可以将背包中的任意装备“炼化”。
- **效果**: 立即获得等同于该装备 **战力值 (`item.combatPower`)** 的金币。这是处理低价值装备和积累财富的核心方式。
- **炼化奇迹**: 此外，每次炼化都有微小几率触发奇迹，产生更高品质的装备（详见《装备系统》）。
- **无确认提示**: 为了提供更流畅的游戏体验，手动炼化装备时不再有额外的确认提示。
