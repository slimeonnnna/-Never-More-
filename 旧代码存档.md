# 旧代码存档

本文档记录了在功能迭代中被替换或移除的旧代码片段，以供参考。

## 敌人追击消耗 (2024-08-01 更新前)

在 `hooks/useGridManager.ts` 中，粘性敌人在追击玩家后，其属性会根据自身的当前值降低1%。

```typescript
// 旧的属性削弱逻辑
const healthReduction = Math.max(1, Math.floor(monster.stats.maxHealth * 0.01));
const attackReduction = Math.max(1, Math.floor(monster.stats.attack * 0.01));
const defenseReduction = Math.max(1, Math.floor(monster.stats.defense * 0.01));

const newMaxHealth = monster.stats.maxHealth - healthReduction;
const newAttack = monster.stats.attack - attackReduction;
const newDefense = monster.stats.defense - defenseReduction;
```

## 神性对属性的增幅 (2024-08-01 更新前)

在 `services/gameLogic.ts` 的 `calculatePlayerStats` 函数中，神性只对攻击力和防御力提供1%的增幅，不影响魔防。

```typescript
// 旧的神性增幅逻辑
const divinityBonus = 1 + (baseStats.divinity * 0.01);
baseStats.attack = Math.round(baseStats.attack * divinityBonus);
baseStats.defense = Math.round(baseStats.defense * divinityBonus);
```
